XOOPS é um sistema de publicação para a criação de sites dinâmicos, é de código livre, lançado sob os termos da licença GNU (GPL), e concebido usando programação PHP orientada a objetos. XOOPS é ideal para a criação de comunidades virtuais, portais, sites de notícias, intranets ou weblogs, de pequena a grande escala. Com milhares de recursos já incorporados, o XOOPS SGC representa a solução ideal para quem não quer depender de terceiros para criar e adminstrar o seu próprio portal.
